API 就绪

后端已运行。前端页面替换此文件即可。

构建前端后，将产物放入 public/ 目录，重新 wrangler deploy。

快速开始

1. npx wrangler d1 create mouse-oj-db

2. 把输出的 ID 填入 wrangler.toml

3. npx wrangler d1 execute mouse-oj-db --local --file=./schema.sql

4. npx wrangler r2 bucket create mouse-oj-assets

5. npx wrangler secret put VJUDGE_COOKIE

6. npx wrangler d1 execute mouse-oj-db --remote --file=./schema.sql

7. npx wrangler deploy
